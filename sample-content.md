## Content Best Practices

### Writing Tips:
- **Clear headlines:** Use descriptive, SEO-friendly titles
- **Strong excerpts:** Write compelling 1-2 sentence summaries
- **Alt text:** Always add alt text to images
- **Categories:** Use 1-3 relevant categories per post

### SEO Optimization:
- **Meta titles:** Keep under 60 characters
- **Meta descriptions:** Keep under 160 characters  
- **Keywords:** Add 3-5 relevant keywords
- **Images:** Optimize file sizes, use descriptive names

### Publishing Workflow:
1. Write in drafts (don't set publishedAt initially)
2. Add images and format content
3. Set SEO fields
4. Set publishedAt when ready to publish
5. Mark featured posts sparingly (1-2 max)

## Organizing Content

Use the Studio's structure:
- **All Posts:** See everything
- **Featured Posts:** Highlight your best content
- **Draft Posts:** Work in progress
- **Authors:** Manage author profiles  
- **Categories:** Organize topics
